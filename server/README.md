# React Back End
- Framework [express](https://expressjs.com/)
- Database [PostgreSQL](https://www.postgresql.org/)
- Cache [Redis](https://redis.io/)  ***-TBD***

## Requirements
- Nodejs
- PostgreSQL server
- Redis server

## Develope
### Initialize dev environment:
```
npm install
```

### [Option] Run Database and Cache on Docker
```
% start docker
docker-compose up -d 
% stop docker
docker-compose down
```
### Run dev environment
```
npm run dev
```
